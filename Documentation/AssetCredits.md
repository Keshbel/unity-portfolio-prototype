# Asset Credits

No copyrighted or pirated assets are included.

The committed playable scene uses Unity primitives, built-in UGUI resources,
manually configured project materials, and a small curated set of CC0
Quaternius models. Complete source packs are intentionally not committed.

## Committed Content

- Unity primitive environment, pickup, generator, extraction-zone, and enemy
  meshes: created in-editor, no external asset license required.
- Simple project materials: created in-editor, no external asset license
  required.
- Audio: none. Legal lightweight audio remains future work.

## Quaternius CC0 Models

Source: Quaternius, <https://quaternius.com/>

License: CC0 1.0 Universal Public Domain Dedication. Copies of the source
license files are committed next to the selected FBX assets.

- Ultimate Modular Sci-Fi Pack: `Door_Double`, `Pipes`, `Wall_2`, and
  `Details_Vent_2`.
- Survival Pack: `FirstAidKit_Hard`, `Radio`, `GasCan`, `Trashcan`, and
  `PropaneTank`.
- Animated Monsters: `Slime`.
